# 📧 Fast Mail Launcher

Bulk email sender using Gmail App Password — hosted on Vercel.

## Files
- `index.html` — Frontend UI
- `style.css` — Styling
- `app.js` — Frontend JS
- `api/send-email.js` — Backend API
- `package.json` — Dependencies
- `vercel.json` — Vercel config

## Deploy
1. GitHub pe repo banao, saari files upload karo
2. vercel.com → New Project → repo select → Deploy
3. Done!

## Gmail App Password
1. Google Account → Security → 2-Step Verification ON
2. App Passwords → Mail → Generate
3. 16-character password use karo app mein
